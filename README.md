# 🧮 BMI Calculator (React)

A simple BMI (Body Mass Index) Calculator built using **React.js**.  
Users can enter height and weight to calculate BMI and view the health category instantly.

## 🚀 Tech Stack
- React.js
- JavaScript
- HTML5
- CSS3

## ✨ Features
- BMI calculation using standard formula
- Input validation
- Instant result (no page reload)
- Reset functionality
- Clean & responsive UI

## ▶️ Run Locally
```bash
npm install
npm start
